📄 Istruzioni per l'uso: Generatore di Foliario

Cos'è questo strumento?

Il Generatore di Foliari Pro è un'applicazione progettata per creare in pochi secondi l'indice dettagliato (Foliario) di un'intera alberatura di cartelle e file.

Il programma scansiona il contenuto, ignora in automatico i file di sistema (es. file nascosti) e genera un report strutturato a livelli, pronto per essere allegato alla documentazione ufficiale.

🚀 1. Primo Avvio

Il programma non richiede installazione.

•	Per utenti Mac (versione .app): Fai doppio clic sull'applicazione. Nota: se il Mac blocca l'apertura perché proviene da uno "sviluppatore non identificato", fai clic destro sull'icona dell'app, scegli "Apri" e conferma.

•	Per utenti Windows (versione .exe): Fai doppio clic sul file eseguibile. Nota: al primo avvio, Windows Defender potrebbe mostrare una schermata blu ("PC protetto da Windows"). Niente panico: clicca su "Ulteriori informazioni" e poi su "Esegui comunque".

🛠️ 2. Come creare un Foliario (Passo dopo passo)

Passo 1: Selezione della Cartella

•	Clicca sul pulsante "📁 1. Seleziona Cartella" nel pannello di destra.

•	Scegli la cartella principale ("Padre") che contiene tutti i documenti da indicizzare. L'albero dei file apparirà al centro dello schermo.

Passo 2: Revisione e Filtro

•	Il programma include automaticamente tutti i file validi.

•	Se noti una cartella o un file che NON deve comparire nel documento finale (es. cartelle di backup o bozze), fai doppio clic sopra di essa.

•	L'elemento diventerà grigio e barrato. Se escludi una cartella, verranno esclusi automaticamente anche tutti i file al suo interno. (Puoi fare di nuovo doppio clic per re-includerli).

•	Usa i pulsanti "➕ Espandi Tutto" e "➖ Comprimi Tutto" per navigare comodamente anche in archivi molto grandi. Tieni d'occhio le "Statistiche" in basso a destra per controllare i totali.

Passo 3: Esportazione

Quando l'elenco è pronto, scegli il formato di esportazione:

•	"📊 3. Esporta Excel (.xlsx)": (Consigliato) Genera un file Excel nativo, già impaginato, con le colonne allargate e le righe delle cartelle evidenziate in azzurro e grassetto per la massima leggibilità.

•	"📄 2. Esporta CSV": Genera un file di testo grezzo (separato da punto e virgola), utile se devi importare i dati in altri software gestionali.

💡 Informazioni incluse nel report

Il Foliario generato distribuirà le cartelle e i file in colonne progressive (Livello 0, Livello 1, Livello 2, ecc.) per mostrare la gerarchia visiva. Inoltre, per ogni file verranno estratti automaticamente:

•	Tipo (Cartella, File o Link/Alias)

•	Peso (Dimensione in KB o MB)

•	Data e Ora di Ultima Modifica

•	Estensione (es. .pdf, .docx)
